Welcome to the Weather App!

This app allows you to get weather information for any city using the OpenWeather API.

Settings

To use this app, you need to set up your OpenWeather API key and URL. You can do this by going to the Settings page.

Dashboard

Once you have set up your API key and URL, you can use the Dashboard to search for the weather in any city.

Login

To log in, click the Login button and select the Google oAuth2 option.

Thank you for using the Weather App! We hope you find it useful.
